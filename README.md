# Biblioteca Comunitária

Projeto desenvolvido utilizando HTML, CSS, JavaScript, FastAPI e SQLite.

## Funcionalidades

* Listar livros
* Buscar livro por ID
* Cadastrar livro
* Atualizar livro
* Excluir livro

## Tecnologias

* HTML
* CSS
* JavaScript
* Python
* FastAPI
* SQLite
* Git

## Execução

Backend:

uvicorn backend.main:app --reload

Frontend:

python -m http.server 5500

A API pode ser acessada em:

/docs

para utilização do Swagger.
